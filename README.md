# Веб-сайт підприємства теплопостачання

## Встановлення

```bash
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

## Структура проекту

- `app.py` - головний файл додатку
- `documents/` - папка для зберігання PDF документів
- `photos/` - папка для зберігання фотографій підприємства
- `.streamlit/config.toml` - конфігурація теми Streamlit
- `config.txt` - файл з інформацією про підприємство
- `contacts.txt` - файл з контактною інформацією

## Налаштування контенту

### 1. Інформація про підприємство

Створіть файл `config.txt` з наступною структурою:

```
NAME:Назва вашого підприємства
DESC:Опис вашого підприємства.

Тут може бути багаторядковий текст.
```

### 2. Контактна інформація

Створіть файл `contacts.txt` з наступною структурою:

```
PHONE:+380 (XX) XXX-XX-XX
EMAIL:email@example.com
ADDRESS:Україна, місто, вулиця, будинок
HOURS:Пн-Пт: 8:00 - 17:00
```

### 3. Документи

Помістіть PDF файли в папку `documents/`:
- `Ліцензія.pdf` - ліцензія на виробництво теплової енергії
- `Тариф.pdf` - тариф на послуги з теплопостачання

**Примітка:** PDF файли не повинні перевищувати 10 MB для коректного відображення в браузері.

### 4. Фотографії

Помістіть фотографії (PNG, JPG, JPEG) в папку `photos/`. Вони будуть автоматично відображені в галереї.
